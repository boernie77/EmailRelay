Automatische SSH-Sicherung

Erstellt regelmäßig ein vollständiges Backup aller Aliases, Benutzer und Domains als ZIP und überträgt es per SFTP auf einen entfernten Server (z. B. Heimserver, NAS, Raspberry Pi).

Einmalige Einrichtung – Schritt für Schritt:

1. Auf dem E-Mail Relay Server: Schlüsselpaar erzeugen:

ssh-keygen -t ed25519 -f /tmp/backup_key -N ""

2. Auf dem E-Mail Relay Server: Public Key anzeigen und kopieren:

cat /tmp/backup_key.pub

3. Auf dem Backup-Server (z. B. Heimserver) den kopierten Public Key eintragen:

mkdir -p ~/.ssh && chmod 700 ~/.ssh echo "HIER_PUBLIC_KEY_EINFÜGEN" >> ~/.ssh/authorized_keys chmod 600 ~/.ssh/authorized_keys

4. Wieder auf dem E-Mail Relay Server: Privaten Schlüssel anzeigen und hier einfügen:

cat /tmp/backup_key

Den gesamten Text von -----BEGIN bis -----END kopieren.

Das Verzeichnis muss bereits existieren und beschreibbar sein.
