Blockieren: Alias bleibt gespeichert, aber deaktiviert. Mails werden still verworfen — der Absender bekommt keine Rückmeldung. Kann jederzeit reaktiviert werden. Auch bei aktiviertem Catch-all wird ein blockierter Alias nicht neu angelegt.

Rotieren: Erstellt einen neuen zufälligen Alias für dieselbe Adresse und deaktiviert den alten. Nützlich wenn ein Alias in Spam-Listen gelandet ist.

Löschen: Entfernt den Alias dauerhaft. Mails an die alte Adresse werden mit "User unknown" abgelehnt — der Absender bekommt eine Bounce-Nachricht. Achtung: Ist Catch-all aktiv, wird der Alias beim nächsten Eingang automatisch neu angelegt — in dem Fall lieber Blockieren verwenden.
