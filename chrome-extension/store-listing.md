# Chrome Web Store Listing – E-Mail Relay

## Name
E-Mail Relay

## Kurzbeschreibung (max. 132 Zeichen)
Alias-Adressen direkt beim Surfen erstellen — schützt deine echte E-Mail-Adresse vor Spam und Tracking.

## Ausführliche Beschreibung (Deutsch)

**Schütze deine echte E-Mail-Adresse mit einem Klick.**

E-Mail Relay ist eine selbst gehostete Datenschutzlösung, vergleichbar mit Firefox Relay oder SimpleLogin — aber vollständig unter deiner eigenen Kontrolle.

Mit dieser Extension erstellst du direkt beim Ausfüllen von Webformularen eine einmalige Alias-Adresse. Registrierungen, Newsletter und Kontaktformulare erhalten nur den Alias — deine echte Adresse bleibt privat.

**So funktioniert es:**
1. Klick auf das Extension-Icon
2. Ziel-Postfach auswählen und Bezeichnung eingeben (z.B. "Amazon")
3. Alias wird erstellt und automatisch ins Formularfeld eingetragen
4. Antworten an den Alias werden an dein echtes Postfach weitergeleitet
5. Unerwünschte Aliases können jederzeit blockiert werden

**Voraussetzungen:**
Diese Extension benötigt einen eigenen E-Mail Relay-Server (selbst gehostet, Open Source auf GitHub). Sie funktioniert **nicht** ohne eigene Serverinstallation.

→ Mehr Informationen und Installationsanleitung: https://github.com/boernie77/EmailRelay

**Datenschutz:**
Die Extension kommuniziert ausschließlich mit deinem eigenen Server. Keine Daten werden an Dritte weitergegeben. Vollständige Datenschutzerklärung: https://api.byboernie.de/privacy
